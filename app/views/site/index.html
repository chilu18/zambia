What is Code & Coffee?

Code & Coffee is an informal get together to write some code in the morning before the real job kicks in. There are no requirements other than the willingness to sit down, pair up, and learn something new. We're going with mornings because the evenings are full enough as it is. Why add to the chaos of user groups, soccer games, and lawn mowing?

If you're interested in joining in, just show up. No invited needed, you don't need to RSVP, you just need a laptop. Coffee not provided, but it's just a few steps away. ;)

Code & Coffee locations are available world wide, check the Locations below to see if there's one near you. If there isn't a location near you, start one! It's fun!

Add a new location

Instructions on how to fork the repository and make pull requests can be found on github.

Once you have the repository forked, open the locations.yml file and follow the pattern of the other locations. Submit a pull request, and VOILA! New location!
